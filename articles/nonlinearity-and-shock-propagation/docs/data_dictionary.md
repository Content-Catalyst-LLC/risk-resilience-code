# Data Dictionary

| Column | Meaning |
|---|---|
| system_name | Name of the system, network, infrastructure corridor, ecological system, public service, or planning unit |
| sector | Sector such as energy, water, health, transport, digital systems, finance, food systems, ecosystems, or governance |
| shock_type | Type of initiating disturbance, such as climate shock, infrastructure outage, cyber incident, public-health surge, financial stress, ecological disturbance, or compound event |
| shock_intensity_index | Initial strength, scale, or severity of the disturbance |
| threshold_proximity_index | How close the system is to a critical threshold or tipping condition |
| network_centrality_index | Whether the affected node is highly connected or systemically important |
| coupling_strength_index | Degree of tight connection and speed of failure transmission |
| feedback_amplification_index | Strength of reinforcing dynamics that intensify the disturbance |
| hidden_stress_index | Accumulated but under-visible vulnerability such as deferred maintenance, debt, ecological degradation, burnout, or institutional strain |
| exposure_inequality_index | Unequal vulnerability across households, workers, communities, regions, or ecosystems |
| buffering_capacity_index | Reserves, wetlands, storage, surge capacity, emergency funds, institutional slack, or other shock absorbers |
| modularity_index | Ability to isolate failure and prevent system-wide spread |
| redundancy_index | Alternate pathways for preserving essential function |
| adaptive_response_index | Capacity to act, learn, reroute, repair, and adjust under changing conditions |
| governance_quality_index | Coordination, legitimacy, accountability, public communication, and decision capacity |

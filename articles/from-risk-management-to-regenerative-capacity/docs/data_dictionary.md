# Data Dictionary

| Column | Meaning |
|---|---|
| system_name | Name of the system, place, institution, watershed, food system, community, or portfolio |
| jurisdiction | Geographic or institutional jurisdiction |
| system_type | System type such as watershed, food system, city, public institution, landscape, infrastructure, or livelihood system |
| risk_management_capacity_index | Hazard assessment, exposure reduction, preparedness, and protective capacity |
| ecological_restoration_index | Soil, water, biodiversity, watershed, landscape, or ecosystem restoration |
| social_capacity_index | Social trust, mutual aid, care systems, community organization, and shared capability |
| institutional_learning_index | Feedback, correction, institutional memory, adaptation, and accountability |
| justice_orientation_index | Attention to structural vulnerability, unequal exposure, marginalized communities, and fair recovery |
| long_term_investment_index | Investment in restoration, adaptation, public capacity, maintenance, and durable systems |
| livelihood_viability_index | Durability and dignity of livelihoods, especially in food, land, rural, and care systems |
| ecological_function_index | Current functionality of ecological processes that support resilience |
| public_trust_index | Public legitimacy, credibility, participation, and institutional reliability |
| adaptive_governance_index | Governance capacity to revise policy, coordinate, and act under uncertainty |
| community_participation_index | Meaningful public, local, Indigenous, worker, farmer, or stakeholder participation |
| regenerative_finance_index | Finance directed toward risk reduction, restoration, adaptation, and long-term renewal |
| depletion_pressure_index | Ecological, social, institutional, or material depletion pressure |
| maladaptation_risk_index | Risk that short-term protection increases long-term vulnerability |
| extractive_pressure_index | Development, finance, production, or governance pressure that draws down future resilience |
| institutional_fatigue_index | Staff burnout, public distrust, undercapacity, and repeated crisis strain |
| chronic_vulnerability_index | Persistent social, ecological, or economic vulnerability that makes shocks more damaging |
| recovery_only_bias_index | Degree to which the system favors post-shock recovery over prevention and renewal |

# Data Dictionary

| Column | Meaning |
|---|---|
| system_name | Name of the system, supply chain, infrastructure network, public service, utility, institution, or planning unit |
| sector | Sector such as supply chains, healthcare, energy, water, public services, transport, finance, food systems, ecosystems, or governance |
| system_type | System category such as lean supply chain, critical infrastructure, hospital network, utility system, public agency, or ecological production system |
| routine_efficiency_index | Ordinary-period cost, throughput, utilization, coordination, and waste-reduction performance |
| protective_slack_index | Buffers, spare capacity, surge capacity, staff margin, inventory, time margin, emergency funds, and fallback room |
| redundancy_index | Multiple credible pathways for preserving essential function |
| modularity_index | Ability to isolate failure and prevent system-wide spread |
| diversity_index | Diversity of suppliers, energy sources, skills, institutions, ecosystems, knowledge systems, or operational pathways |
| feedback_monitoring_index | Sensors, audits, inspections, worker voice, community reporting, stress indicators, and transparent performance data |
| repair_capacity_index | Maintenance, spare parts, workforce skill, restoration planning, local repair capacity, and recovery capability |
| governance_quality_index | Coordination, accountability, public trust, decision capacity, emergency planning, and institutional legitimacy |
| tight_coupling_index | Speed and strength of failure propagation across connected components |
| single_point_dependence_index | Reliance on one supplier, node, route, platform, facility, authority, or technical component |
| overload_index | Chronic use near or beyond safe operating capacity |
| deferred_maintenance_index | Accumulated infrastructure, institutional, workforce, ecological, or technical repair deficits |
| hidden_risk_transfer_index | Degree to which risk is shifted to workers, households, small suppliers, communities, ecosystems, or future budgets |

# Data Dictionary

| Column | Meaning |
|---|---|
| institution_or_system | Name of the public institution, agency, service system, or governance arrangement |
| jurisdiction | Geographic or institutional jurisdiction |
| public_function | Public function such as healthcare, benefits, courts, schools, emergency response, water, transport, or public administration |
| essential_function_clarity_index | Clarity about which services, rights, and functions must continue under stress |
| capacity_margin_index | Slack, surge capacity, reserves, and minimum viable service capacity |
| dependency_visibility_index | Visibility into suppliers, infrastructure, digital systems, staff roles, and interagency dependencies |
| workforce_resilience_index | Staffing, training, safety, rotation, surge capacity, and institutional memory |
| digital_resilience_index | Cyber readiness, backups, interoperability, recovery time, and manual fallback |
| legal_authority_clarity_index | Clarity of emergency powers, duties, thresholds, and review mechanisms |
| coordination_capacity_index | Cross-agency, cross-scale, and cross-sector coordination capacity |
| equity_protection_index | Ability to preserve access for vulnerable communities under stress |
| public_trust_index | Legitimacy, credible communication, and public confidence |
| recovery_capacity_index | Ability to restore essential functions after disruption |
| backup_systems_index | Backup facilities, data, power, staffing, communications, and manual procedures |
| mutual_aid_capacity_index | Capacity to draw on neighboring jurisdictions, partners, community organizations, or shared public systems |
| fiscal_reserve_capacity_index | Emergency funding, reserve capacity, and budgetary flexibility |
| learning_capacity_index | Ability to learn from stress tests, incidents, audits, and after-action reviews |
| accountability_mechanism_index | Audits, public reporting, oversight, reform tracking, and remedy mechanisms |
| overload_pressure_index | Demand spikes, staff fatigue, resource strain, and cognitive load |
| institutional_fragmentation_index | Siloed authority, duplicated responsibility, and coordination gaps |
| hidden_fragility_index | Untested assumptions, unknown dependencies, and invisible failure thresholds |
| vendor_dependency_index | Dependence on external vendors, platforms, contractors, or suppliers |
| compound_stress_exposure_index | Exposure to simultaneous, cascading, or interacting stressors |

# Data Dictionary

| Column | Meaning |
|---|---|
| planning_system | Name of the scenario matrix, shock library, foresight process, or resilience planning system |
| jurisdiction | Geographic or institutional jurisdiction |
| system_type | System type such as city, agency, infrastructure network, health system, food system, public institution, watershed, or regional plan |
| scenario_coverage_index | Breadth and relevance of plausible futures considered |
| shock_library_quality_index | Completeness, usefulness, governance, and structure of the shock library |
| shock_specificity_index | Operational specificity of shock descriptions, including duration, severity, affected functions, and pathways |
| compound_risk_coverage_index | Coverage of compound, cascading, and interacting shocks |
| essential_function_mapping_index | Mapping between shocks and essential services, rights, functions, or systems |
| dependency_mapping_index | Visibility of dependencies across infrastructure, suppliers, institutions, digital systems, and jurisdictions |
| trigger_readiness_index | Clarity of early-warning indicators and action triggers |
| threshold_clarity_index | Clarity about when system strain becomes unacceptable or failure-prone |
| equity_integration_index | Integration of unequal exposure, vulnerable groups, and distributional protection |
| community_validation_index | Extent to which affected communities validate shocks, scenarios, assumptions, and interpretations |
| stress_test_linkage_index | Degree to which scenarios and shocks are used to stress-test actual plans, budgets, and investments |
| governance_ownership_index | Clarity of institutional owners responsible for updates, actions, and corrections |
| update_cadence_index | Frequency and quality of review cycles after new evidence, events, or exercises |
| action_linkage_index | Degree to which findings trigger decisions, resources, reform, or preparedness actions |
| adaptive_decision_capacity_index | Ability to revise decisions as evidence, scenarios, and conditions change |
| compound_risk_exposure_index | Exposure to simultaneous, interacting, or cascading shocks |
| blind_spot_gap_index | Missing shocks, omitted groups, weak data, or unexamined assumptions |
| stale_assumption_risk_index | Risk that outdated assumptions remain embedded in plans or scenarios |
| scenario_theater_risk_index | Risk that scenario work performs foresight without changing decisions |
| untested_fragility_index | Hidden weaknesses that have not been tested through exercises, scenarios, or real events |

## Suggested Input Dataset

Expected file:

```text
scenario_matrix_shock_library_panel.csv
```

Each `*_index` field should be normalized from 0 to 1.

Higher values should mean more of the named property. For example:

- `scenario_coverage_index`: higher = broader coverage of plausible futures
- `shock_specificity_index`: higher = more operationally useful shock definitions
- `compound_risk_exposure_index`: higher = greater exposure to compound or cascading shocks
- `scenario_theater_risk_index`: higher = greater risk that scenario work does not affect decisions

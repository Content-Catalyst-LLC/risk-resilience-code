# Model Notes

This article uses conceptual resilience-design scoring models.

The scores are not objective measurements of resilience. They are diagnostic tools for making design assumptions visible.

## Core Concepts

- Optimization-fragility score
- Resilience capacity score
- Resilience-adjusted design risk

## Interpretation

A system can have strong normal-period performance and still carry high resilience-design risk.

The model is most useful for comparative diagnosis, stress testing, governance review, and scenario discussion.

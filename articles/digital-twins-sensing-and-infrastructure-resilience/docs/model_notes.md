# Model Notes

This article uses conceptual digital twin resilience scoring models.

The scores are not objective measurements of digital twin performance. They are diagnostic tools for making implementation assumptions visible.

## Core Concepts

- **Digital twin resilience contribution**: estimates how much a twin improves sensing, data quality, model validation, decision integration, dependency visibility, maintenance, climate adaptation, equity coverage, and governance.
- **Digital twin implementation risk**: estimates risk from cyber exposure, platform dependency, model uncertainty, weak validation, weak data quality, weak governance, weak public accountability, and poor equity coverage.
- **Resilience readiness score**: combines resilience contribution, lower implementation risk, service continuity relevance, governance capacity, and public accountability.

## Interpretation

A digital twin can be visually sophisticated and still have weak resilience readiness.

The model is most useful for comparative diagnosis, infrastructure planning, resilience review, and governance discussion.

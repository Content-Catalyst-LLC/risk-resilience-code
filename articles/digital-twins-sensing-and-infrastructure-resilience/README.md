# Digital Twins, Sensing, and Infrastructure Resilience

This article directory supports the Risk & Resilience article:

**Digital Twins, Sensing, and Infrastructure Resilience**

The workflows model how digital twins, sensing systems, data quality, model validation, decision integration, cybersecurity, governance, and public accountability shape infrastructure resilience.

## Analytical Focus

The article examines:

- Digital twin resilience contribution
- Sensing coverage
- Data quality and timeliness
- Model validation
- Decision integration
- Dependency visibility
- Predictive maintenance usefulness
- Climate adaptation usefulness
- Service continuity relevance
- Cyber risk
- Platform dependency
- Model uncertainty
- Equity coverage
- Governance capacity
- Public accountability

## Suggested Input Dataset

Expected file:

```text
digital_twin_infrastructure_panel.csv
```

Each `*_index` field should be normalized from 0 to 1.

Higher values should mean more of the named property. For example:

- `sensing_coverage_index`: higher = better sensing coverage
- `data_quality_index`: higher = stronger data quality
- `cyber_risk_index`: higher = greater cyber risk
- `governance_capacity_index`: higher = stronger governance capacity

# Model Notes

This article uses conceptual resilience-dashboard blind-spot scoring models.

The scores are not objective measurements of dashboard truth. They are diagnostic tools for making dashboard assumptions, blind spots, and governance risks visible.

## Core Concepts

- **Dashboard strength**: estimates indicator coverage, usability, data quality, update frequency, disaggregation, equity visibility, uncertainty transparency, action linkage, community validation, governance accountability, threshold clarity, and stress-performance integration.
- **Blind-spot risk**: estimates risk from false precision, proxy dependence, missing data, aggregation loss, scale-boundary error, dashboard theater, and overall blind-spot severity.
- **Dashboard actionability**: estimates whether dashboard indicators are linked to thresholds, ownership, corrective action, update cycles, and governance decisions.
- **Equity-adjusted dashboard reliability**: combines dashboard strength, actionability, lower blind-spot risk, equity visibility, and community validation.
- **Dashboard reliability gap**: compares dashboard strength against blind-spot risk.

## Interpretation

A dashboard can look polished and still be weak if it hides uncertainty, lacks disaggregation, depends on proxies, excludes vulnerable groups, or fails to trigger action.

The model is most useful for dashboard review, governance audits, indicator redesign, resilience scorecard critique, and measurement-quality assessment.

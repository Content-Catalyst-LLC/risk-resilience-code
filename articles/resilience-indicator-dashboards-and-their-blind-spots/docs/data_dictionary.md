# Data Dictionary

| Column | Meaning |
|---|---|
| dashboard_name | Name of the resilience dashboard, scorecard, monitoring tool, or indicator system |
| jurisdiction | Geographic or institutional jurisdiction |
| dashboard_type | Dashboard type such as city, project, public institution, infrastructure, climate adaptation, disaster risk, community, or portfolio |
| indicator_coverage_index | Breadth and relevance of indicator coverage across resilience domains |
| dashboard_usability_index | Clarity, navigability, interpretability, and practical usefulness |
| data_quality_index | Accuracy, completeness, timeliness, representativeness, and reliability of data |
| update_frequency_index | How often indicators are updated relative to decision needs |
| disaggregation_strength_index | Ability to inspect results by geography, group, sector, system component, or vulnerability category |
| equity_visibility_index | Visibility of unequal exposure, protection, recovery, access, and participation |
| uncertainty_transparency_index | Clarity about uncertainty, confidence, proxy status, data gaps, and assumptions |
| action_linkage_index | Degree to which indicators connect to ownership, triggers, resources, and corrective action |
| community_validation_index | Extent to which affected communities can validate, challenge, or interpret dashboard findings |
| governance_accountability_index | Strength of ownership, review, public accountability, and reform mechanisms |
| threshold_clarity_index | Clarity about when dashboard conditions require action or signal system failure |
| stress_performance_integration_index | Integration of stress tests, scenarios, recovery curves, and performance-under-stress evidence |
| false_precision_risk_index | Risk that dashboard numbers appear more exact than evidence supports |
| proxy_dependence_index | Degree of reliance on indirect indicators that may not capture actual resilience performance |
| missing_data_risk_index | Risk from absent, outdated, undercounted, incomplete, or biased data |
| aggregation_loss_index | Loss of interpretive detail when scores aggregate across people, places, or system components |
| scale_boundary_error_index | Risk that dashboard boundaries exclude important dependencies, groups, or external fragilities |
| dashboard_theater_risk_index | Risk that reporting and visualization substitute for actual governance action |
| blind_spot_severity_index | Overall severity of known or suspected dashboard blind spots |

# Data Dictionary

| Column | Meaning |
|---|---|
| ecosystem_name | Name of the ecosystem, landscape, seascape, protected area, watershed, restoration area, or monitoring unit |
| jurisdiction | Geographic or institutional jurisdiction |
| ecosystem_type | Ecosystem category such as forest, wetland, grassland, river, coastal, marine, agricultural landscape, or urban ecosystem |
| genetic_diversity_index | Adaptive variation within populations or species |
| species_diversity_index | Diversity of species within the ecological community |
| functional_diversity_index | Range of ecological roles performed by organisms |
| habitat_connectivity_index | Connectivity across habitats, corridors, watersheds, seascapes, and ecological networks |
| ecosystem_integrity_index | Condition, structure, composition, and functional integrity of the ecosystem |
| adaptive_capacity_index | Capacity of the system to respond to changing conditions |
| governance_quality_index | Quality of ecological governance, protection, management, accountability, and adaptive learning |
| community_stewardship_index | Strength of Indigenous, local, community, or participatory stewardship |
| fragmentation_pressure_index | Habitat fragmentation and isolation pressure |
| pollution_pressure_index | Pollution pressure affecting air, water, soil, organisms, and ecological processes |
| invasive_pressure_index | Pressure from invasive alien species or disruptive biological invasions |
| extraction_pressure_index | Pressure from overharvest, exploitation, land conversion, or unsustainable resource use |
| climate_stress_index | Climate-related pressure such as heat, drought, fire, flood, ocean warming, or changing seasons |
| monitoring_gap_index | Weakness in biodiversity evidence, species monitoring, ecological data, or uncertainty documentation |

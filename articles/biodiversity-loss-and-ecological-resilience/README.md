# Biodiversity Loss and Ecological Resilience

This article directory supports the Risk & Resilience article:

**Biodiversity Loss and Ecological Resilience**

The workflows model how genetic diversity, species diversity, functional diversity, habitat connectivity, ecosystem integrity, adaptive capacity, governance quality, community stewardship, fragmentation pressure, pollution pressure, invasive pressure, extraction pressure, climate stress, and monitoring gaps shape biodiversity-supported ecological resilience.

## Analytical Focus

The article examines:

- Biodiversity loss
- Ecological resilience
- Genetic diversity
- Species diversity
- Functional diversity
- Response diversity
- Ecological redundancy
- Habitat connectivity
- Ecological memory
- Ecosystem integrity
- Climate risk and biodiversity
- Ecosystem services
- Food, water, health, and livelihood systems
- Thresholds and regime shifts
- Fragmentation and degradation
- Indigenous and local knowledge
- Biodiversity governance and measurement

## Suggested Input Dataset

Expected file:

```text
biodiversity_ecological_resilience_panel.csv
```

Each `*_index` field should be normalized from 0 to 1.

Higher values should mean more of the named property. For example:

- `species_diversity_index`: higher = greater species diversity
- `habitat_connectivity_index`: higher = stronger ecological connectivity
- `fragmentation_pressure_index`: higher = greater habitat fragmentation pressure
- `monitoring_gap_index`: higher = weaker evidence coverage

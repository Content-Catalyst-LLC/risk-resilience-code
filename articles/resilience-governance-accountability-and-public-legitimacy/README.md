# Resilience Governance, Accountability, and Public Legitimacy

This article directory supports the Risk & Resilience article:

**Resilience Governance, Accountability, and Public Legitimacy**

The workflows model how adaptive capacity, accountability, public legitimacy, transparency, participation, coordination, learning, institutional memory, justice orientation, oversight, remedy, correction, capture risk, trust erosion, fragmentation, and vulnerable exposure shape legitimate resilience governance.

## Analytical Focus

The article examines:

- Resilience governance quality
- Accountability capacity
- Public legitimacy
- Adaptive capacity
- Transparency
- Participation and contestation
- Coordination across sectors and scales
- Learning capacity
- Institutional memory
- Justice orientation
- Oversight strength
- Remedy availability
- Correction capacity
- Institutional fragmentation
- Capture risk
- Trust erosion
- Vulnerability exposure
- Systemic risk exposure

## Suggested Input Dataset

Expected file:

```text
resilience_governance_legitimacy_panel.csv
```

Each `*_index` field should be normalized from 0 to 1.

Higher values should mean more of the named property. For example:

- `adaptive_capacity_index`: higher = stronger adaptive capacity
- `accountability_capacity_index`: higher = stronger accountability
- `public_legitimacy_index`: higher = stronger public legitimacy
- `capture_risk_index`: higher = greater capture risk
- `trust_erosion_index`: higher = greater erosion of public trust

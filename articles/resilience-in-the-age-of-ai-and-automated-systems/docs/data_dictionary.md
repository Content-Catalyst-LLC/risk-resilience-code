# Data Dictionary

| Column | Meaning |
|---|---|
| system_name | Name of the AI-enabled system being assessed |
| sector | Sector such as finance, government, infrastructure, logistics, healthcare, education, communications, or public services |
| ai_use_case | AI use case, such as monitoring, classification, prediction, eligibility, anomaly detection, planning, or decision support |
| model_reliability_index | Validity, reliability, robustness, and performance across expected contexts |
| monitoring_capacity_index | Post-deployment monitoring, drift detection, incident tracking, and review capacity |
| oversight_capacity_index | Meaningful human review, escalation, and override capacity |
| governance_strength_index | Institutional authority, documentation, auditability, and governance capability |
| explainability_index | Degree to which outputs can be interpreted or explained for the use case |
| contestability_index | Ability to challenge, appeal, explain, or correct AI-influenced decisions |
| fallback_capacity_index | Manual procedures, redundant systems, and recovery pathways |
| data_quality_index | Accuracy, completeness, relevance, and reliability of data |
| bias_management_index | Capacity to detect, measure, mitigate, and monitor harmful bias |
| privacy_protection_index | Privacy-preserving design, data minimization, and safeguards |
| cyber_resilience_index | Resistance to compromise, manipulation, outage, and cyber disruption |
| concentration_risk_index | Dependency on shared vendors, models, platforms, or data sources |
| automation_dependence_index | Degree of overreliance on automated decision processes |
| drift_risk_index | Exposure to model drift, distribution shift, or degraded performance |
| opacity_risk_index | Risk from lack of interpretability, explainability, or diagnostic visibility |
| public_accountability_index | Transparency, oversight, public-interest safeguards, and accountability |
| service_criticality_index | Importance of the AI-enabled system to essential services or high-stakes decisions |

# Data Dictionary

| Column | Meaning |
|---|---|
| dataset_name | Name of the dataset, dashboard source, indicator table, model input, or evidence artifact |
| jurisdiction | Geographic or institutional jurisdiction |
| data_domain | Data domain such as hazard, exposure, vulnerability, infrastructure, recovery, finance, public health, social protection, cyber, or community evidence |
| provenance_completeness_index | Completeness of source, creator, collection, transformation, ownership, and downstream-use records |
| metadata_quality_index | Quality of definitions, units, temporal coverage, spatial coverage, licenses, limitations, and update details |
| lineage_clarity_index | Ability to trace data from raw records to final indicators, models, dashboards, or reports |
| audit_evidence_index | Availability of inspectable records, approvals, logs, inputs, outputs, and review evidence |
| reproducibility_index | Ability to recreate outputs from documented data, code, parameters, and versions |
| data_quality_index | Accuracy, completeness, timeliness, consistency, representativeness, and validity |
| version_control_index | Strength of versioning for datasets, code, models, parameters, documentation, and outputs |
| chain_of_custody_index | Integrity of evidence handling, access control, timestamps, storage, and tamper resistance |
| equity_coverage_index | Visibility of vulnerable groups, marginalized communities, and unequal data coverage |
| community_validation_index | Extent to which affected communities can validate, challenge, or interpret data claims |
| privacy_safeguard_index | Strength of privacy protections, minimization, consent, access limits, and disclosure controls |
| security_control_index | Strength of cybersecurity and protected-evidence controls |
| responsible_owner_index | Clarity of data ownership, maintenance responsibility, approval authority, and stewardship |
| missingness_gap_index | Missing data, undercounted groups, incomplete coverage, or absent records |
| opacity_risk_index | Risk that methods, ownership, assumptions, or outputs cannot be explained or inspected |
| undocumented_transformation_index | Risk from unrecorded cleaning, joins, imputation, weighting, aggregation, or model transformations |
| stale_data_risk_index | Risk that outdated records are used for current resilience claims |
| sensitive_data_exposure_risk_index | Risk that auditability or data sharing exposes sensitive personal, institutional, or security information |
| audit_gap_index | Gap between evidence needed for responsible review and evidence available for verification |

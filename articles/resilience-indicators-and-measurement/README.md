# Resilience Indicators and Measurement

This article directory supports the Risk & Resilience article:

**Resilience Indicators and Measurement**

The workflows model how capacity indicators, asset indicators, process indicators, outcome indicators, equity protection, service continuity, recovery performance, adaptive learning, institutional capacity, ecological condition, social protection, financial protection, vulnerability, exposure, distributional inequality, measurement uncertainty, data quality gaps, and false precision risk shape resilience measurement.

## Analytical Focus

The article examines:

- Resilience indicators
- Capacity, asset, process, and outcome indicators
- Service continuity
- Recovery performance
- Adaptive learning
- Institutional capacity
- Ecological condition
- Social protection
- Financial protection
- Vulnerability
- Exposure
- Distributional inequality
- Equity-adjusted resilience
- Measurement uncertainty
- Data quality gaps
- False precision risk
- Indicator confidence
- Composite score limitations

## Suggested Input Dataset

Expected file:

```text
resilience_indicators_panel.csv
```

Each `*_index` field should be normalized from 0 to 1.

Higher values should mean more of the named property. For example:

- `capacity_index`: higher = stronger resilience capacity
- `outcome_index`: higher = stronger resilience outcomes
- `vulnerability_index`: higher = greater vulnerability
- `measurement_uncertainty_index`: higher = greater uncertainty in the measurement framework

# Data Dictionary

| Column | Meaning |
|---|---|
| system_name | Name of the system, institution, place, portfolio, community, or project being measured |
| jurisdiction | Geographic or institutional jurisdiction |
| system_type | System type such as city, project, institution, infrastructure, household, ecosystem, watershed, or portfolio |
| capacity_index | Preparedness, response, learning, adaptive governance, coordination, and capability |
| asset_index | Physical, financial, natural, human, social, and institutional resources |
| process_index | Planning, participation, maintenance, accountability, risk governance, and learning processes |
| outcome_index | Reduced losses, improved continuity, recovery, adaptation, and reduced future vulnerability |
| equity_protection_index | Distributional fairness, inclusion, access, and protection for vulnerable groups |
| service_continuity_index | Ability to preserve essential services under stress |
| recovery_performance_index | Speed, quality, completeness, and fairness of recovery after disturbance |
| adaptive_learning_index | Ability to learn from shocks, revise assumptions, and improve future preparedness |
| institutional_capacity_index | Administrative, legal, technical, workforce, and coordination capacity |
| ecological_condition_index | Ecological function, restoration, redundancy, diversity, and system health |
| social_protection_index | Social safety nets, household protection, care systems, and shock-responsive support |
| financial_protection_index | Insurance, reserves, fiscal capacity, risk pools, and pre-arranged finance |
| vulnerability_index | Susceptibility to harm due to social, physical, institutional, economic, or ecological conditions |
| exposure_index | People, assets, institutions, services, or ecosystems located in hazard-prone conditions |
| distributional_inequality_index | Unequal protection, recovery, access, or resilience outcomes across groups or places |
| measurement_uncertainty_index | Uncertainty caused by proxies, assumptions, weighting, and incomplete evidence |
| data_quality_gap_index | Missing, weak, outdated, biased, or uneven data used for measurement |
| false_precision_risk_index | Risk that a score appears more exact or authoritative than the evidence supports |

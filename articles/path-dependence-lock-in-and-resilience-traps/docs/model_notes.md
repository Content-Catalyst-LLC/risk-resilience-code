# Model Notes

This article uses conceptual path dependence, lock-in, resilience trap, and transformation readiness scoring models.

The scores are not objective certifications of system transformation or trap status. They are diagnostic tools for making sunk cost, infrastructure rigidity, institutional inertia, incumbent power, social dependence, technological incompatibility, ecological feedback, alternative capacity, adaptive governance, public legitimacy, justice transition support, and reversibility visible.

## Core Concepts

- **Lock-in pressure**: the force keeping a system tied to an incumbent pathway.
- **Transformation capacity**: the capacity to move toward a more viable, just, and sustainable pathway.
- **Resilience trap risk**: the risk that a system remains resilient in preserving an undesirable state.
- **Transformation readiness**: the degree to which alternatives, governance, legitimacy, justice, and reversibility are present.
- **Escape gap**: the difference between transformation capacity and lock-in pressure.
- **Just transition support**: protections for workers, households, communities, and vulnerable groups during pathway change.

## Interpretation

A system can be persistent because it is healthy and adaptive, or because it is trapped in an undesirable equilibrium. This model helps distinguish persistence worth preserving from persistence that reproduces harm.

The model is most useful for infrastructure transition planning, climate adaptation review, just transition strategy, social-ecological trap analysis, resilience governance, policy redesign, and sustainable systems transformation.

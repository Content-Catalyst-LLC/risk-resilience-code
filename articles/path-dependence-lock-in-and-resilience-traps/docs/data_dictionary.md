# Data Dictionary

| Column | Meaning |
|---|---|
| system_name | Name of the system, pathway, infrastructure network, institution, sector, or social-ecological system |
| sector | Sector such as energy, water, transport, housing, agriculture, fisheries, governance, public health, ecosystems, or finance |
| pathway_type | Type of pathway such as carbon lock-in, infrastructure lock-in, institutional lock-in, social-ecological trap, poverty trap, or maladaptive adaptation pathway |
| sunk_cost_index | Investment already committed to the incumbent pathway |
| infrastructure_rigidity_index | Durability and inflexibility of physical systems that constrain future options |
| institutional_inertia_index | Rules, routines, mandates, budgets, and professional norms that preserve the current pathway |
| incumbent_power_index | Political, financial, legal, and organizational power defending the existing system |
| social_dependence_index | Jobs, identity, household routines, public revenue, and community livelihoods tied to the pathway |
| technological_incompatibility_index | Standards, platforms, skills, and complementary assets that make alternatives harder to adopt |
| ecological_feedback_index | Ecological degradation or feedback processes that reinforce vulnerability or reduce recovery options |
| alternative_capacity_index | Viable alternatives, pilots, infrastructure, skills, institutions, and financing for a new pathway |
| adaptive_governance_index | Institutional ability to learn, coordinate, revise assumptions, and act across sectors |
| public_legitimacy_index | Public trust, participation, transparency, and perceived fairness of transformation pathways |
| justice_transition_index | Protections for workers, households, vulnerable communities, and affected regions during change |
| reversibility_index | Degree to which harmful commitments can be modified, phased out, redesigned, or exited |
